# Show Modal

This page provides information for using the `showModal()` framework function to open a Modal widget.

## Signature

```javascript
showModal(modalName: string): Promise
```

### Arguments

#### Modal Name

<dd>

_(string)_ Name of the Modal widget to open.

</dd>

### Example

```javascript
{{ showModal("Modal1") }}
```
