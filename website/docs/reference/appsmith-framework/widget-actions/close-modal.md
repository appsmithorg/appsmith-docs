# Close Modal

This page provides information for using the `closeModal()` framework function to open a Modal widget.

## Signature

```javascript
closeModal(modalName: string): Promise
```

### Arguments

#### Modal Name

<dd>

_(string)_ Name of the Modal widget to close.

</dd>

### Example

```javascript
{{ closeModal("Modal1") }}
```
