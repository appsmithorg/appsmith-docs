# Connect Git Repository

Appsmith allows you to connect your applications to any remote Git repository. This capability enables you to integrate your Appsmith apps with your preferred version control system. 

You can connect to Git using any provider of your choice with a Remote SSH URL. The following guides are tailored to popular Git providers such as GitHub, Bitbucket, GitLab, and Azure.
