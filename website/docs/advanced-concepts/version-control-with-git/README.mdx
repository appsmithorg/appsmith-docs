# Version Control With Git

Version control with Git in Appsmith offers a streamlined solution for managing your applications. 

Version Control with Git in Appsmith not only allows multiple developers to collaborate seamlessly on the same application but also enables efficient tracking of changes, rollback of alterations, and collaboration through Git branches.

## Core features


- **Version Control Integration:** Connects with Git hosting providers (GitHub, GitLab, Bitbucket, Azure and more) for version control.

- **Granular Commit Control:** Granular Git commit modal allows precise tracking and rollback of changes.

- **Branch Management:** Facilitates collaboration using Git branches.

- **Automated Source File Management:** Converts Appsmith application components into source files for Git.

- **Auto-Commit:** Automatically commits changes related to DSL version upgrades, ensuring your app stays updated with the latest features without manual intervention.

- **Secure Secret Management:** Stores configuration securely while excluding sensitive data from version control.

- **CI/CD Integration:** Supports Continuous Integration and Continuous Delivery workflows, enabling automatic updates to the master/main branch and integration with platforms like GitHub Actions and Bitbucket Pipelines.
