# Instance Management
One of the key features of Appsmith is its instance management capabilities. This page is a gateway to various management tasks you can perform on your Appsmith instance. 

You can manage your instance by performing:
* Backups for Appsmith instance and database
* Updating Appsmith to a new version
* Scheduling a maintenance window for auto updates
* Restoring backups. 

This page provides information to set up and maintain your Appsmith installation.
