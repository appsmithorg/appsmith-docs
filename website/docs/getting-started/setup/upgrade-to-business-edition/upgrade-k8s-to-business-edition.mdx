Follow the steps below to upgrade Kubernetes Community edition to Business edition:

<ul class="circle"> 
<li> Uninstall the community edition Helm chart by running the following command:

         helm uninstall appsmith
</li>
<li>Run the following commands to add and deploy the new Helm chart:

         helm repo add appsmith-ee https://helm-ee.appsmith.com
    helm repo update
    helm install appsmith appsmith-ee/appsmith-ee -n <namespace> -f values.yaml
 </li>
<li> Sign-in to your instance again. Upon successful login, you see a screen where you can enter the license key and activate the instance.
</li>

</ul>
