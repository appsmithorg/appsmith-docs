Follow the steps below to upgrade Kubernetes Community edition to Business edition:

1. Uninstall the community edition Helm chart by running the following command:

    ```javascript
    helm uninstall appsmith
    ```
2. Run the following commands to add and deploy the new Helm chart:

    ```javascript
    helm repo add appsmith-ee https://helm-ee.appsmith.com
    helm repo update
    helm install appsmith appsmith-ee/appsmith-ee -n <namespace> -f values.yaml
    ```
3. Sign-in to your instance again. Upon successful login, you see a screen where you can enter the license key and activate the instance.
