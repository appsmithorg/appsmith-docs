Follow the steps listed below to install Appsmith on docker:

1. Download and place the `docker-compose.yml` file in the Appsmith installation folder or run the cURL command.

    ```bash
    curl -L <ADD_NEW_BITLY_LINK_TO_DOWNLOAD -o $PWD/docker-compose.yml
    ```

2. Start the Docker container using the command `docker-compose up -d ` (You may need to run with `sudo`).

3. Check the Docker logs for the message `Appsmith is Running!` using the command `docker logs -f appsmith` and confirm that the Appsmith container is ready. Server logs can be found at `stacks/logs/backend/backend.log`.

4. Create or sign in to your Appsmith account, and enter your license key to activate the instance.
