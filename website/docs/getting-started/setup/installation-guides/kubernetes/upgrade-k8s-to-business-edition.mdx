# Upgrade Kubernetes to business edition

<HideElements/>

Follow the steps below to upgrade Kubernetes Community edition to Business edition:

1. Uninstall the community edition helm charts by running the following command:

```javascript
helm uninstall appsmith
```
2. Install and delpoy the business edition helm charts by running the following commands:

```javascript
helm repo add appsmith-ee https://helm-ee.appsmith.com
helm repo update
helm install appsmith appsmith-ee/appsmith-ee -n <namespace> -f values.yaml
```
5. Sign-in to your instance again. Upon successful login, a screen will be shown where you can enter your license key and activate the instance.
