# Upgrade to Commercial Edition

Appsmith offers Commercial and Community Editions.

* **Community Edition** is the open source edition of Appsmith.

* **Commercial Edition** provides both free and paid plans making it easy for users to upgrade or downgrade. The Commercial edition is available on self-hosted Appsmith. For more information on the features and pricing of each plan, see [Pricing](https://www.appsmith.com/pricing).
