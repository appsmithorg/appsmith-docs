# Upgrade to Commercial Edition

Appsmith offers both **Community Edition** and **Commercial Edition**.

* **Community Edition**: The open-source version of Appsmith.
* **Commercial Edition**: Provides both free and paid plans, offering flexibility to upgrade or downgrade as needed. The Commercial Edition is available for self-hosted Appsmith instances. For detailed information on features and pricing of each plan, visit the [Pricing](https://www.appsmith.com/pricing) page.
