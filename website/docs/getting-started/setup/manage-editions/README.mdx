# Manage Editions

Appsmith offers Commercial and Community Editions. 

* **Community Edition** is an open source edition of Appsmith..
* **Commercial Edition** that includes both free and paid plans making it easy for users to upgrade or downgrade. For more information, see [Pricing](https://www.appsmith.com/pricing).
