# Admin Settings



The Admin Settings section allows administrators to configure and control instance-wide settings in Appsmith. These settings govern how your entire workspace operates—from user authentication and access control to branding, infrastructure configuration, and licensing.

Only users with Instance Administrator privileges can access the Admin Settings panel.
If you are unable to view these settings but believe you should have access, please contact your organization’s Appsmith administrator to be granted the appropriate role.

:::caution important
Settings configured via Environment Variables override any settings configured via the Admin Settings UI.
:::
