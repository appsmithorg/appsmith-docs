# Download 

## Signature

```text
download(data: any, fileName: string, fileType?: string) -> void
```

#### Arguments

| **Argument Name** | **Description** |
| :--- | :--- |
| **data** |  Data that would be downloaded |
| **fileName** |  Name of the file to be downloaded |
| **fileType** |  Type of the file to be downloaded |
